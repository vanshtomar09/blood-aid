## 📖 About

Blood Aid is a full-stack web application designed to simplify blood donation and blood bank management. It enables users to search blood availability, manage donors, and visualize inventory through an intuitive dashboard.

> **Note:** This project is currently under active development and uses demo data to showcase application workflows.

## ✨ Features

- Blood donor management
- Blood inventory dashboard
- Recipient search
- Hospital dashboard
- Responsive UI
- Authentication (planned)
- Real-time updates (planned)

## 🛠 Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Firebase
- Genkit AI
